매일매일 학습 콘텐츠

매일 할수 있도록 체계적 분량으로 구성된 학습 콘텐츠로 영어 말하기, 듣기, 읽기, 쓰기의 기본부터 고급 단계까지 실력이 탄탄해 집니다.

  • [English play] Emma & Guardians
    #신나는 #영어 #말하기
    #읽기 #듣기
    • 대상 : Phonics를 끝낸 학습자
    • 수준 : 기초 회화
    • 구성 : 10개 챕터, 100개 에피소드

    실물 교재와 같이 학습합니다.

    My Work에서 결과물을 확인합니다.

  • [English play] Magic Adventures
    #신나는 #영어 #말하기
    #읽기 #쓰기 #듣기
    • 대상 : 기초 회화 학습 경험자
    • 수준 : 기초 + 기본 회화
    • 구성 : 18개 챕터, 132개 에피소드

    실물 교재와 같이 학습합니다.

    My Work에서 결과물을 확인합니다.

  • Story Play
    #내가 만들어가는 #이솝우화
    #듣기 #읽기 #쓰기
    • 내가 만들어가며, 새로 쓰는 이솝우화
    • 이솝 우화의 결말을 직접 선택합니다.
    • 상상력을 길러주는 읽기에 푹 빠집니다.

    실물 교재에 내가 스토리를 직접 쓰고 꾸며볼수 있습니다.

  • Popcorn
    #입에서 #팡팡터지는 #단어학습
    #듣기 #읽기 #쓰기
    • 1,200개 단어가 즐거워서 나도 모르게 습득합니다.
    • 아이들에 친숙한 45개의 주제별로 단어가 분류되어 있습니다.
    • 단어 난이도 선택이 가능합니다.
  • Audio Play
    #읽어주는 #흥미만점
    #이야기책 #듣기
    • 30개 이상의 재미있는 스토리의 오디오북이 탑재되어 있습니다.
    • 잠자기 전, 아침에 일어나서 틀어주기
    • 방해 금지 모드로 취침 시간에는 자동 정지됩니다.
  • Al Show Room 스피킹 버디의 기본 제공 콘텐츠 외에도 교육 회사, 학습 콘텐츠 크리에이터들과 제휴하여 다양한 효과높은 학습 콘텐츠을 AI 쇼룸을 통해 제공합니다. *8월 경 추가 콘텐츠 제공 예정

친구같이 언제나 즐겁게 학습하는 콘텐츠

편한 친구같이 언제나 원할 때 재미있게 영어를 학습할 수 있습니다.

  • Dictionary
    #1,200개 단어 #주제별 분류
    #이미지 #단어 #듣기
    • 1,200개 단어의 탑재
    • 팝콘에 들어간 단어와 연계
    • 주제를 선택하여 간편하게 단어 찾기
    • 지속적인 추가 단어 업데이트
    • AI 음성으로 단어 발음
  • Memory Game
    #즐거운 #단어 #학습
    #단어 게임 #팝콘
    • Popcorn에서 배운 단어를 재미있게 메모리 게임으로 복습
    • Level 1 ~ 4 단계까지 세분화
    • 이미지와 단어의 매칭으로 평생 기억
  • Cast World
    #Roleplay #주인공
    #말하기 #표현력 #영작
    • 내가 주인공이 되어 영어로 말해보기
    • 재미있는 꾸밈 효과가 들어간 영상, 음원 제작하기
    • 영어 문장에 대한 기본기 형성

    캐스트 월드는 제공 예정입니다.

  • Pop TV
    #매일매일 #전송되는 #큐레이션
    #콘텐츠 #듣기 #재미
    • 주제, 학습 과정별로 필요한 엄선된 영어 영상 콘텐츠가 매일 도착
    • 유튜브의 다른 영상은 재생 불가하므로 마음껏 시청해도 OK!
  • Radio On
    #영어로 #라디오 #듣기
    • 아리랑 영어방송 채널 듣기
    • 듣기 노출 효과 상승
    • 잠자기 전에 틀어주면 좋습니다.

    영어 라디오 채널 추가 예정
